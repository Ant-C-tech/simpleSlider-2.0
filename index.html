Use simpleSlider 2.0 at your project:

  • Add Bootstrap 4 to your HTML.
  • Add simpleSlider_2_0.js to your HTML.
  • Add your own JavaScript file (after simpleSlider_2_0.js !) to your HTML.
  • Create slider-container on any place you need. It will be DIV with class = "container galery (your name for slider-container) text-center mt-3 mb-3" (not necessary).
  • Add to your js-file some collection of images (slides) like this: 
        const imgCollection = {
        slide1: ['https://picsum.photos/800', '#'],
        slide2: ['https://picsum.photos/801', '#'],
        slide3: ['https://picsum.photos/802', '#'],
        slide4: ['https://picsum.photos/803', '#'],
        slide5: ['https://picsum.photos/804', '#'],
    }
  • Every slide contains "src" and "alt" atributes.
  • Slider initialisation in your js-file: 
        let slider = new Slider({
         container: '.galery',               
         auto: true,               
         time: 3000               
    },                
    imgCollection)
